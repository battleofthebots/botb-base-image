# BOTB Base
This repository defines the base image of the Battle of the Bots competition

The latest version of `botb-base` is based on the `ubuntu 20.04` and contains the following packages:

### from apt
- python3-pip
- build-essential
- default-jre
- nmap
- netcat-openbsd
- socat
- iproute2
- iputils-ping
- openssh-client
- telnet
- ftp
- curl
- wget

### from pip
- paramiko
- requests
- aiohttp
- urllib3
- pyyaml
- scapy
- impacket
- pwntools
- cryptography
- pycryptodome
